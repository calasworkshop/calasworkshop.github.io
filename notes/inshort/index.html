Numbas kort fortalt

Numbas er et nettbasert system som lager digitale tester for matematikk og andre fag. Testene lages i en editor server som er åpen for alle. Numbas er også et lagersted for tester som andre har laget og som du kan kopiere og arbeide videre med.

Questions and Exams

Numbas Editor lager enkeltoppgaver (questions) som så kan samles sammen til en test (exam). I en exam kan det settes opp om studentene kan gjøre flere forsøk, hoppe mellom spørsmål, tidsrammer, se tips underveis, løsningsforslag og andre parametre. En exam kan altså brukes til egentrening og selvstudium - eller til en avsluttende eksamen.

Oppgavetypene i Numbas er tallsvar, tekstsvar, flervalg, paring, algebraiske uttrykk, matriser - alle med mulighet for randomiserte tall og uttrykk. Det kan også settes inn figurer og videoer.

To typer tester

Noen LMS'er kan importere tester som er pakket inn i SCORM eller LTI format og kjøre disse som om de tilhører LMS'et. Dermed kan studentenes svar og resultater bli lagret i LMS'ets eget statusregnskap. Numbas kan lage slike utgaver av testene for SCORM format, og LTI format er under utvikling. Ikke alle LMS'er følger standardene fullt ut.

Når du lager oppgaver i Numbas og setter disse sammen til en test kan du altså velge å laste den ned som SCORM/LTI pakke eller som en Standalone (nettside) utgave. Du kan også laste ned selve kildekoden til testene som en Source - og dermed ta egne kopier av testene.

Alt foregår hos studenten

Det som skiller Numbas fra andre systemer er at testene lages som enkle nettsider (html-dokumenter) som hentes fra filservere via en link, fra LMS'er eller minnepinner. I en slik nettside ligger både oppgaveteksten og et et system som lager tilbakemelding om resultatet. En Numbas test er altså ikke avhengig av at et LMS porsjonerer ut oppgaver til studentene og holder rede på resultater.

Dette gjør Numbas til et lettvekt system i forhold til andre som er avhengig av kommunikasjon med en server som viser spørsmål og prosesserer svar.

Numbas Editor

Numbas utvikles ved Newcastle University. De tilbyr alle å bruke deres editor server til å lage oppgaver. Siden Numbas er et system med åpen kildekode er det også mulig å sette opp egne editor servere og lage oppgaver der. Det kan være aktuelt ved uttesting av systemet - eller til utvikling av egne løsninger.

Fordelen med å bruke Numbas Editor ved Newcastle University er tilgangen til andres oppgaver. Ved redigering av oppgaver vil vi også forhåndsvise testene fra denne serveren. Når vi så skal la studentene bruke testene må vi altså først laste ned disse i SCORM eller Standalone format og legge de ut i LMS eller andre filservere (Dropbox?).

Oppgavedatabase

De fritt tilgjengelige oppgavene i Numbas Editor ved Newcastle University er inndelt etter fag, emner eller skoleslag. I tillegg kan de være merket med egne tagger som beskriver innholdet. Det finnes ikke en katalog eller liste med alle tilgjengelige oppgaver - de må søkes opp i Browse the public data base der søket kan innsnevres etter fag/emne/skoleslag, tagger, forfatter, status eller lisensrettigheter.

Nye oppgaver eller eksamen er i utgangspunktet bare tilgjengelig for de som redigerer den, men kan gjøres Public etter å ha blitt merket med fag/emne og lisensinformasjon. Alle oppgaver tilhører et prosjekt. En institusjon kan lage prosjektgrupper som holder orden på egne oppgaver.

LaTeX og enlinje syntaks

Oppgavene lages i en tekstbasert editor som kan gjøre enkle formateringer av teksten. Matematiske uttrykk i oppgaveteksten tastes inn som LaTeX kode som automatisk blir tolket og vist som formeluttrykk. Studentene svarer med enlinje syntax - som i de aller fleste matteapper, CAS-kalkulatorer og regneark. Svarene blir tolket fortløpende og vist som LaTeX uttrykk - eller med meldinger som hjelper til riktig inntasting. Svarene aksepteres ikke før de er matematisk korrekte.
